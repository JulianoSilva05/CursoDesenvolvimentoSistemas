Tópicos da Aula

  • Formulário Multipart: enctype="multipart/form-data".
  • $_FILES: O array de arquivos enviados.
  • Upload: move_uploaded_file(). Movendo da pasta temporária para o destino.
  • Relacionamentos SQL: Tabelas relacionadas (ex: Produtos pertencem a Categorias).
  • JOIN: Consultando dados de múltiplas tabelas.

Exemplo Prático: Upload Simples


// Form: <input type="file" name="foto">

if (isset($_FILES['foto'])) {
    $arquivo = $_FILES['foto'];
    $destino = 'uploads/' . $arquivo['name'];
    
    if (move_uploaded_file($arquivo['tmp_name'], $destino)) {
        echo "Upload realizado!";
    }
}

Atividade Prática (2h)

⚠️ Importante: Esta atividade é INDIVIDUAL. Você pode consultar apenas o material do professor e links autorizados. Não saia desta tela.

Para Todos (Essencial)

Crie um formulário simples que permite ao usuário enviar uma imagem (JPG/PNG). Salve a imagem em uma pasta "uploads" e exiba a imagem na tela após o envio.

Desafio (Avançado)

Integre isso ao cadastro de produtos. Adicione um campo de foto. Salve o caminho da foto no banco de dados e exiba a imagem do produto na listagem (tabela).
