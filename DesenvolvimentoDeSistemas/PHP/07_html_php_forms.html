Integração HTML e PHP

Formulários e Validação

Aula 07

Duração Total: 4 Horas

2h Teoria + 2h Prática

Tópicos da Aula

  • Formulários HTML: A porta de entrada de dados.
  • Métodos HTTP: GET (URL, visível) vs POST (Body, oculto).
  • Superglobais: $_GET e $_POST.
  • Validação no Back-end: Nunca confie no usuário! Usando empty(), isset(), filter_var().
  • XSS Básico: Protegendo a saída com htmlspecialchars().

Exemplo Prático: Recebendo Dados


<!-- index.php -->
<form method="POST" action="recebe.php">
    <input type="text" name="nome">
    <button>Enviar</button>
</form>

<!-- recebe.php -->
<?php
if (isset($_POST['nome'])) {
    $nome = htmlspecialchars($_POST['nome']);
    echo "Olá, $nome!";
}
?>

Atividade Prática (2h)

⚠️ Importante: Esta atividade é INDIVIDUAL. Você pode consultar apenas o material do professor e links autorizados. Não saia desta tela.

Para Todos (Essencial)

Crie um formulário simples com campos para "Nome" e "Idade". Ao clicar em enviar, o script PHP deve mostrar uma mensagem: "Olá [Nome], você tem [Idade] anos.".

Desafio (Avançado)

Crie um formulário de "Contato" (Nome, Email, Mensagem). Valide no PHP: Todos os campos são obrigatórios e o Email deve ser válido (use filter_var($email, FILTER_VALIDATE_EMAIL)). Se passar, exiba "Mensagem enviada!". Se não, exiba os erros.
