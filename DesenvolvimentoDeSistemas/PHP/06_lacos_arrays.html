Laços de Repetição e Arrays

Estruturas de Dados

Aula 06

Duração Total: 4 Horas

2h Teoria + 2h Prática

Tópicos da Aula

  • Laços (Loops): Repetindo ações. for, while.
  • Arrays: Listas de dados. Indexados (0, 1, 2...) e Associativos (chaves nomeadas).
  • Foreach: O laço perfeito para Arrays.
  • Funções: function nome() { ... }. Parâmetros e Retorno.
  • Manipulação de Arrays: count(), array_push(), implode().

Exemplo Prático: Foreach


$produtos = [
    ["nome" => "Notebook", "preco" => 3000],
    ["nome" => "Mouse", "preco" => 50]
];

foreach ($produtos as $prod) {
    echo "Produto: " . $prod['nome'] . " - R$ " . $prod['preco'];
    echo "
";
}

Atividade Prática (2h)

Para Todos (Essencial)

Crie um array com nomes de 5 filmes. Use um laço foreach para exibir cada nome dentro de uma lista HTML (<ul><li>...).

Desafio (Avançado)

Crie uma função chamada tabuada($numero). Ela deve receber um número e imprimir a tabuada dele de 1 a 10 usando um laço for. Chame a função para os números 5 e 9.
