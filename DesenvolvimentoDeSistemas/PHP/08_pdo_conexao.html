PDO e Segurança

Aula 08

Duração Total: 4 Horas

2h Teoria + 2h Prática

Tópicos da Aula

  • Extensões PHP: MySQLi vs PDO. Por que usar PDO? (Suporte a múltiplos bancos).
  • DSN (Data Source Name): Configurando host, dbname e charset.
  • Conexão: Usando try...catch para capturar falhas.
  • Segurança: O que é SQL Injection? Como hackers invadem sistemas mal feitos.
  • Prepared Statements: A vacina contra SQL Injection.

Exemplo Prático: Conectando


$host = 'localhost';
$db   = 'escola';
$user = 'root';
$pass = '';

try {
    $pdo = new PDO("mysql:host=$host;dbname=$db", $user, $pass);
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    echo "Conectado!";
} catch(PDOException $e) {
    echo "Erro de conexão: " . $e->getMessage();
}

Atividade Prática (2h)

Para Todos (Essencial)

Crie um banco de dados chamado loja_virtual no MySQL (phpMyAdmin). Crie um script conexao.php que se conecta a ele. Se der certo, mostre "Sucesso". Se a senha estiver errada, mostre uma mensagem de erro amigável.

Desafio (Avançado)

Implemente a conexão usando o padrão Singleton (Classe estática que retorna sempre a mesma instância da conexão), evitando abrir múltiplas conexões desnecessárias.
