Introdução ao Flexbox

O que é?

O Flexbox (Flexible Box Layout) é um modelo de layout unidimensional projetado para distribuir espaço entre itens em uma interface e alinhavá-los de forma poderosa.

Ele resolve problemas antigos do CSS, como centralizar elementos verticalmente ou fazer colunas terem a mesma altura.

Quando Usar?

  • ✅ Menus de Navegação: Alinhar links lado a lado.
  • ✅ Centralização: Colocar algo exatamente no meio da tela.
  • ✅ Cards e Listas: Fazer itens se ajustarem ao tamanho da tela.
  • ❌ Layouts de Página Inteira (Complexos): Para isso, prefira o CSS Grid (que veremos depois).

Analogia: Pense no Flexbox como um colar de pérolas. Você pode esticar o fio (espaçar as pérolas), empurrá-las todas para um canto ou deixá-las soltas no meio.

Guia Visual do Flexbox (CSS-Tricks)

Usando como referência o melhor guia da internet: CSS-Tricks Flexbox Guide

Container (Pai)

display: flex;
Transforma o elemento em um container flexível. Seus filhos viram itens flex.

Direção (flex-direction)

Define o Eixo Principal.
row (linha - padrão)
column (coluna - ideal para mobile)

Alinhamento (A Mágica)

Justify Content (Eixo Principal)

Alinha os itens horizontalmente (se estiver em row).

  • flex-start: Início (Esquerda)
  • center: Centro
  • space-between: Espaço igual entre eles

Align Items (Eixo Transversal)

Alinha os itens verticalmente (se estiver em row).

  • stretch: Estica (Padrão)
  • center: Centraliza no meio da altura

Flex Wrap (Quebra de Linha)

Por padrão, o Flexbox tenta colocar tudo na mesma linha, espremendo os itens se necessário.

O flex-wrap: wrap; permite que os itens "quebrem" para a linha de baixo se não houver espaço.

.galeria {
  display: flex;
  flex-wrap: wrap;
}

Como criar Links (Tag <a>)

Links são a base da internet. Eles conectam páginas. A tag usada é a <a> (Anchor/Âncora).

Estrutura Básica


<a href="destino.html">Clique Aqui</a>

href: O endereço para onde o link vai (pode ser um site externo ou um arquivo local).

Atributos Úteis

  • target="_blank": Abre em nova aba.
  • class="botao": Para estilizar como botão no CSS.

Responsividade e Mobile First

Mobile First significa criar o site pensando primeiro na tela pequena do celular e depois ajustar para telas grandes.


/* 1. Estilo padrão (Celular) */
.container {
    flex-direction: column; /* Um item embaixo do outro */
}

/* 2. Media Query (Tablet/PC - Telas acima de 768px) */
@media (min-width: 768px) {
    .container {
        flex-direction: row; /* Um item ao lado do outro */
    }
}

Dica de Mestre: Usando IA (Trae)

A Inteligência Artificial é sua co-piloto. Use o Trae para gerar ideias de CSS, mas sempre entenda o código.

✅ O que pedir (Simples)

"Trae, crie um CSS para deixar este botão com bordas arredondadas, cor azul e sombra suave quando passar o mouse."

Isso ajuda você a aprender novas propriedades como border-radius, box-shadow e :hover.

❌ O que evitar (Agora)

"Trae, faça o site inteiro pra mim."

Se você fizer isso, não vai treinar sua lógica. Use a IA para estilizar e tirar dúvidas de sintaxe.

Exercício com IA: Tente pedir ao Trae: "Gere um gradiente de fundo bonito usando azul e roxo para o meu body." e aplique no seu código!
