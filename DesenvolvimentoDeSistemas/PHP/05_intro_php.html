Introdução ao PHP

Variáveis e Controle

Aula 05

Duração Total: 4 Horas

2h Teoria + 2h Prática

Tópicos da Aula

  • O que é PHP? Linguagem de script do lado do servidor (Backend).
  • Sintaxe Básica: <?php ... ?>.
  • Variáveis: Começam com $ (ex: $nome). Tipagem dinâmica.
  • Tipos de Dados: String, Integer, Float, Boolean, Array, Null.
  • Operadores: Aritméticos (+, -, *, /) e Comparação (==, !=, >, <).
  • Estruturas de Controle: if, else, elseif, switch.

Exemplo Prático: Decisão


<?php
  $idade = 17;
  $temHabilitacao = false;

  if ($idade >= 18 && $temHabilitacao) {
      echo "Pode dirigir!";
  } elseif ($idade >= 18 && !$temHabilitacao) {
      echo "Maior de idade, mas sem CNH.";
  } else {
      echo "Menor de idade.";
  }
?>

Atividade Prática (2h)

⚠️ Importante: Esta atividade é INDIVIDUAL. Você pode consultar apenas o material do professor e links autorizados. Não saia desta tela.

Para Todos (Essencial)

Crie um script PHP (pode ser num arquivo index.php) que define duas variáveis: $nota1 e $nota2. Calcule a média. Se a média for >= 7, exiba "Aprovado". Caso contrário, "Reprovado". Teste mudando os valores.

Desafio (Avançado)

Crie um "Verificador de Ano Bissexto". Regra: É bissexto se for divisível por 4. Exceto se for divisível por 100, a menos que seja divisível por 400. Exiba a mensagem correta para o ano definido na variável.
