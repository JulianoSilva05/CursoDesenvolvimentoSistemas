Aula 04

Duração Total: 4 Horas

Tópicos da Aula

  • JavaScript no Navegador: Onde colocar o script?
  • Lógica Básica: Variáveis (let, const), Tipos de dados, Operações matemáticas.
  • Funções: Criando blocos de código reutilizáveis.
  • DOM (Document Object Model): Acessando o HTML pelo JS (document.querySelector, getElementById).
  • Eventos: Interagindo com o usuário (onclick, addEventListener).

Exemplo Prático: Interatividade


<h1 id="titulo">Olá Mundo</h1>
<button onclick="mudarTexto()">Clique</button>

<script>
function mudarTexto() {
    let elemento = document.querySelector("#titulo");
    elemento.innerText = "Texto Alterado!";
    elemento.style.color = "blue";
}
</script>
