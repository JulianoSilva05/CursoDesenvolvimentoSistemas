CRUD PHP: Update e Delete

Editando e Excluindo

Aula 11

Duração Total: 4 Horas

2h Teoria + 2h Prática

Tópicos da Aula

  • SQL UPDATE: Atualizando registros existentes (Cuidado com o WHERE!).
  • SQL DELETE: Removendo registros (Cuidado redobrado!).
  • Passagem de ID: Usando links como excluir.php?id=10.
  • $_GET['id']: Recuperando qual registro alterar/excluir.
  • Preenchimento: Carregar dados atuais no formulário de edição.

Exemplo Prático: Exclusão


// excluir.php
$id = $_GET['id'];
$sql = "DELETE FROM usuarios WHERE id = :id";
$stmt = $pdo->prepare($sql);
$stmt->bindValue(':id', $id);
$stmt->execute();

header("Location: lista.php"); // Redireciona de volta

Atividade Prática (2h)

Para Todos (Essencial)

Na listagem de produtos, adicione um link "Excluir" ao lado de cada item. Ao clicar, deve chamar excluir.php, remover o item do banco e voltar para a lista.

Desafio (Avançado)

Crie a funcionalidade de "Editar". Link "Editar" leva para editar.php?id=X. O formulário deve vir preenchido com os dados atuais. Ao salvar, faz um UPDATE no banco.
