Melhores Práticas

Aula 14

Duração Total: 4 Horas

2h Teoria + 2h Prática

Tópicos da Aula

  • Organização: Separar lógica de visualização. Pastas: assets, includes, pages.
  • Includes: include, require, require_once. (Header e Footer).
  • Hash de Senhas: Nunca salve senhas em texto puro! Use password_hash() e password_verify().
  • Segurança Web: Proteção contra XSS (Sanitização) e CSRF (Tokens).

Exemplo Prático: Hash de Senha


// Cadastrando
$senha = "minhasenha123";
$hash = password_hash($senha, PASSWORD_DEFAULT);
// Salva $hash no banco (ex: $2y$10$...)

// Logando
$senhaDigitada = "minhasenha123";
$hashDoBanco = "..."; // Vindo do SELECT

if (password_verify($senhaDigitada, $hashDoBanco)) {
    echo "Senha correta!";
}

Atividade Prática (2h)

Para Todos (Essencial)

Reorganize seu projeto. Crie arquivos header.php e footer.php e inclua-os em todas as páginas. Implemente o hash de senha no cadastro de usuários.

Desafio (Avançado)

Implemente um sistema de mensagens de erro/sucesso usando Sessão (Flash Messages). Ex: Ao cadastrar, redirecionar para a lista e mostrar "Sucesso" apenas uma vez.
