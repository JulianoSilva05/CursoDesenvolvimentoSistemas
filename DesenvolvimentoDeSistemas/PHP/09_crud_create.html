CRUD PHP: Create

Inserindo Registros

Aula 09

Duração Total: 4 Horas

2h Teoria + 2h Prática

Tópicos da Aula

  • CRUD: Create, Read, Update, Delete.
  • SQL INSERT: Revisão (INSERT INTO tabela (col1, col2) VALUES (v1, v2)).
  • Prepared Statements: Usando placeholders (:nome, :email) para segurança.
  • Bind Value: Vinculando os valores reais aos placeholders.
  • Execute: Rodando a query e verificando erros.

Exemplo Prático: Inserindo Usuário


$sql = "INSERT INTO usuarios (nome, email) VALUES (:n, :e)";
$stmt = $pdo->prepare($sql);
$stmt->bindValue(':n', $_POST['nome']);
$stmt->bindValue(':e', $_POST['email']);

if ($stmt->execute()) {
    echo "Usuário cadastrado!";
} else {
    echo "Erro ao cadastrar.";
}

Atividade Prática (2h)

Para Todos (Essencial)

Crie uma tabela produtos (id, nome, preco). Crie um formulário HTML e um script PHP que recebe os dados e insere um novo produto no banco usando PDO e Prepared Statements.

Desafio (Avançado)

Antes de inserir, verifique se já existe um produto com o mesmo nome no banco. Se existir, exiba um erro: "Produto já cadastrado". Se não, insira normalmente.
