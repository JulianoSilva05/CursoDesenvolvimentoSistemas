Projeto Final e Deploy

Finalização da Aplicação

Aula 15

Duração Total: 4 Horas

2h Revisão/Ajustes + 2h Apresentação

Tópicos da Aula

  • Revisão Geral: Do HTML ao Banco de Dados.
  • Checklist Final: Testar cadastro, edição, exclusão, login, logout, upload.
  • Hospedagem: Servidores Web (Apache/Nginx).
  • Deploy: Como publicar seu site na internet (ex: InfinityFree, 000webhost).
  • Apresentação: Mostrando o projeto funcional.

Checklist de Deploy


1. Exportar Banco de Dados (.sql)
2. Configurar arquivos de conexão (mudar localhost para o IP do servidor)
3. Upload de arquivos via FTP (FileZilla) ou Gerenciador de Arquivos
4. Importar .sql no phpMyAdmin do servidor
5. Testar tudo online!

Atividade Prática (2h)

Para Todos (Essencial)

Finalize o seu Sistema Completo (CRUD + Login + Upload). Corrija bugs, melhore o layout com CSS/Bootstrap. Prepare-se para mostrar ao professor.

Desafio (Avançado)

Crie uma documentação (README.md) explicando como instalar seu projeto. Se possível, faça o deploy real em um servidor gratuito e gere o link público.

Fim do Curso!

Parabéns por completar o módulo de PHP.

Você agora é capaz de criar aplicações web dinâmicas com banco de dados.
