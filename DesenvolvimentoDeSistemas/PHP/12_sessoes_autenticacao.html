Aula 12

Duração Total: 4 Horas

2h Teoria + 2h Prática

Tópicos da Aula

  • Stateless: O HTTP não guarda estado. Precisamos de Sessões.
  • Sessões PHP: session_start() e $_SESSION. Dados no servidor.
  • Cookies: Dados no navegador do usuário.
  • Fluxo de Login: Verificar credenciais -> Criar sessão.
  • Controle de Acesso: Verificar sessão em cada página restrita.
  • Logout: session_destroy().

Exemplo Prático: Protegendo Página


// painel.php
session_start();

// Se não existir a variável 'usuario' na sessão...
if (!isset($_SESSION['usuario'])) {
    header("Location: login.php"); // Manda pro login
    exit;
}

echo "Bem-vindo, " . $_SESSION['usuario'];

Para Todos (Essencial)

Crie uma página de Login (usuario e senha). Valide se a senha é "1234". Se for, crie uma sessão e redirecione para painel.php. O painel deve ser protegido (ninguém entra sem logar).

Desafio (Avançado)

Implemente o "Logout" (botão Sair). E crie um cookie "usuario_email" para preencher o campo de email automaticamente quando o usuário voltar à página de login (Recurso "Lembrar-me").
