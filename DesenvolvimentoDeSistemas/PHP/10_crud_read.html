CRUD PHP: Read

Listagem de Dados

Aula 10

Duração Total: 4 Horas

2h Teoria + 2h Prática

Tópicos da Aula

  • SQL SELECT: Revisão (SELECT * FROM tabela).
  • Query vs Prepare: query() para consultas simples sem parâmetros.
  • Fetch All: Trazendo todos os resultados de uma vez (fetchAll).
  • Fetch Assoc: Trazendo como array associativo (nomes das colunas).
  • Renderização: Misturando HTML e PHP para criar tabelas dinâmicas.

Exemplo Prático: Listar Usuários


$sql = "SELECT * FROM usuarios";
$stmt = $pdo->query($sql);
$lista = $stmt->fetchAll(PDO::FETCH_ASSOC);

foreach ($lista as $u) {
    echo $u['nome'] . " - " . $u['email'] . "<br>";
}

Atividade Prática (2h)

Para Todos (Essencial)

Crie uma página listar_produtos.php. Conecte ao banco e exiba todos os produtos cadastrados na aula anterior dentro de uma <table> HTML bonita (use CSS/Bootstrap).

Desafio (Avançado)

Adicione um formulário de busca no topo da página. Ao digitar um nome e filtrar, a lista deve exibir apenas os produtos que contenham aquele termo (Use WHERE nome LIKE :termo).
