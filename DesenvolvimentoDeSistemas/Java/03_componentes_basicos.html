JComponent: O Pai de Todos

Quase todos os componentes visuais do Swing (botões, listas, tabelas) herdam de JComponent.

Isso significa que TODOS eles ganham superpoderes de graça:

Tooltips (Dicas de Tela)

Basta uma linha para adicionar uma explicação ao passar o mouse.

botao.setToolTipText("Clique para Salvar");

Bordas (Borders)

Todo JComponent pode ter uma borda personalizada.

botao.setBorder(BorderFactory.createLineBorder(Color.RED));

Atalhos de Teclado (Key Bindings)

Você pode associar teclas (ex: F1, Ctrl+S) a ações do componente.

Pluggable Look & Feel

Eles sabem se desenhar conforme o tema do sistema.

Eventos: O Coração da Interatividade

No Swing, quando o usuário faz algo (clica, digita), o componente dispara um Evento.

Para reagir a esse evento, você precisa de um Listener (Ouvinte).

A Receita do Bolo:

  1. Crie o componente (ex: botão).
  2. Crie um Listener (uma classe que implementa ActionListener).
  3. Registre o Listener no componente (botao.addActionListener(...)).
JButton botao = new JButton("Clique-me"); // Usando Lambda (Java 8+) - Jeito Moderno botao.addActionListener(e -> { System.out.println("Botão clicado!"); JOptionPane.showMessageDialog(null, "Olá Mundo!"); });

Action: Indo além do Listener

Imagine que você tem um comando "Salvar".

O usuário pode salvar clicando no Botão da Toolbar, no Menu Arquivo > Salvar ou apertando Ctrl+S.

Em vez de escrever o código 3 vezes, você cria uma Action.

Action salvarAction = new AbstractAction("Salvar", iconeDiskette) { public void actionPerformed(ActionEvent e) { salvarArquivo(); // Lógica centralizada } }; // Agora conectamos a MESMA ação em lugares diferentes JButton btnToolbar = new JButton(salvarAction); JMenuItem itemMenu = new JMenuItem(salvarAction);

Vantagem: Se você desativar a ação (salvarAction.setEnabled(false)), o botão e o menu ficam cinzas automaticamente!

Teclado e Foco

Para interagir com o teclado, um componente precisa ter o Foco (estar selecionado).

KeyListener (O jeito antigo/básico)

Ouve cada tecla pressionada. Bom para jogos ou controle fino.

campo.addKeyListener(...)

Key Bindings (O jeito Swing)

Associa uma tecla a uma Action. É melhor porque funciona mesmo se o foco estiver em outro lugar da janela (dependendo da configuração).
