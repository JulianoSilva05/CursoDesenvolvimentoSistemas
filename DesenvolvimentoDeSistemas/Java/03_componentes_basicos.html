Interagindo com o Usuário

Conceitos Fundamentais (Teoria)

JButton

Botão de clique padrão.

JTextField

Campo de entrada de texto de uma linha.

Eventos

Ações do usuário (clique, digitação) geram Eventos.

Listeners

Interfaces que 'ouvem' e reagem aos eventos.

Aprofundamento Técnico

1. ActionListener (Lambda)

Desde o Java 8, usamos expressões Lambda para eventos, deixando o código muito mais limpo.

btn.addActionListener(e -> System.out.println("Clicou!"));

2. Hierarquia JComponent

Quase todos os componentes herdam de JComponent, ganhando métodos como setBorder, setToolTipText e setBackground.

Exemplos de Implementação

Analise o código abaixo para entender a aplicação:

JPanel painel = new JPanel(); JTextField campo = new JTextField(20); JButton btn = new JButton("Copiar"); // Ao clicar, pega o texto do campo btn.addActionListener(e -> { String texto = campo.getText(); JOptionPane.showMessageDialog(null, "Você digitou: " + texto); }); painel.add(campo); painel.add(btn);

Atividade Prática (1 Hora)

Desafio: Crie um 'Mini Chat'. Uma janela com um JTextField e um JButton 'Enviar'. Ao clicar, mostre o texto digitado em um System.out.println.
