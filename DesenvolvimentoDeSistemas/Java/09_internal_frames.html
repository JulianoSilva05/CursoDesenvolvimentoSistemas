Aula 09: Aplicações MDI

JDesktopPane, JInternalFrame

Aula 09

Duração: 3 Horas (2h Teoria + 1h Prática)

Múltiplas Janelas Internas

Conceitos Fundamentais (Teoria)

DesktopManager

Gerencia o comportamento das janelas internas (minimizar, arrastar).

JInternalFrame

Propriedades: Resizable, Closable, Maximizable, Iconifiable.

Foco

Como lidar com qual janela interna está ativa (setSelected).

Cascata/Lado a Lado

Algoritmos para organizar as janelas automaticamente (precisa ser implementado manualmente).

Aprofundamento Técnico

1. O Conceito MDI

Popularizado pelo Windows 3.1/95 (Excel, Word antigos), onde tudo roda dentro de uma janela mãe. Útil para softwares de gestão (ERP) que precisam de várias telas abertas simultaneamente.

2. Configuração da Janela Interna

O construtor é chave: new JInternalFrame(Titulo, Resizable?, Closable?, Max?, Icon?). Se você passar 'false' em closable, o usuário não consegue fechar a janela!

Exemplos de Implementação

Analise o código abaixo para entender a aplicação:

JDesktopPane desktop = new JDesktopPane(); this.setContentPane(desktop); // Criando janela filha JInternalFrame relatorio = new JInternalFrame("Relatório", true, true, true, true); relatorio.setSize(300, 300); relatorio.setLocation(50, 50); relatorio.add(new JLabel("Dados aqui...")); relatorio.setVisible(true); // Essencial! desktop.add(relatorio);

Atividade Prática (1 Hora)

Desafio: Crie um 'Mini Sistema Operacional'. Tenha um Menu 'Abrir'. Ao clicar, crie uma nova JInternalFrame. Permita abrir várias janelas e movê-las pela tela.
