Dominando a Exibição de Dados Estáticos

Conceitos Fundamentais (Teoria)

JLabel Básico

Exibição de texto simples e imagens. Componente não editável.

HTML Rendering

O Swing permite usar tags HTML (, ,

) dentro de Labels.

Posicionamento

Controle fino de alinhamento vertical e horizontal (ícone vs texto).

Acessibilidade

Uso de setLabelFor() para vincular labels a campos de texto (mnemônicos).

Aprofundamento Técnico

1. Renderização HTML

O JLabel é capaz de renderizar HTML 3.2. Isso permite criar layouts complexos dentro de um único componente, como listas coloridas ou textos multilinhas.

new JLabel("<html><h1>Título</h1><p>Parágrafo</p></html>");

2. Mnemônicos e Acessibilidade

Atalhos de teclado (Alt+Letra) podem focar campos relacionados.

JLabel lblNome = new JLabel("Nome:"); lblNome.setDisplayedMnemonic('N'); // Alt+N foca o campo lblNome.setLabelFor(txtNome);

Exemplos de Implementação

Analise o código abaixo para entender a aplicação:

// Exemplo Completo: Label Rico ImageIcon icon = new ImageIcon("user.png"); JLabel label = new JLabel("Usuário: Admin
Online", icon, JLabel.LEFT); // Configuração Visual label.setIconTextGap(15); // Espaço entre ícone e texto label.setVerticalTextPosition(JLabel.CENTER); label.setHorizontalTextPosition(JLabel.RIGHT); label.setBorder(BorderFactory.createEtchedBorder()); label.setToolTipText("Status do usuário atual");
