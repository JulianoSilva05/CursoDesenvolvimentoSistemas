Aula 13: Seletor de Arquivos e Cores

JFileChooser, FileFilter, JColorChooser

Aula 13

Duração: 3 Horas (2h Teoria + 1h Prática)

Acesso ao Sistema de Arquivos

Conceitos Fundamentais (Teoria)

FileFilter

Restringe os arquivos visíveis (ex: apenas .txt).

SelectionMode

Arquivos apenas, Diretórios apenas, ou ambos.

Preview

Adicionar um painel de pré-visualização ao FileChooser (Avançado).

ColorChooser

Recuperar a cor selecionada (Color) para aplicar em componentes.

Aprofundamento Técnico

1. Filtrando Arquivos

Para melhorar a experiência do usuário, use FileNameExtensionFilter.

chooser.setFileFilter(new FileNameExtensionFilter("Imagens JPG e PNG", "jpg", "png"));

2. Salvando vs Abrindo

Use showSaveDialog() para salvar. A lógica é a mesma, mas o botão muda para 'Salvar' e o comportamento de sobrescrever arquivo pode ser tratado.

Exemplos de Implementação

Analise o código abaixo para entender a aplicação:

JButton btnCor = new JButton("Mudar Cor de Fundo"); btnCor.addActionListener(e -> { Color cor = JColorChooser.showDialog(this, "Escolha", Color.WHITE); if (cor != null) { getContentPane().setBackground(cor); } });

Atividade Prática (1 Hora)

Desafio: Crie um 'Mini Editor de Texto'. Um TextArea, um botão 'Abrir' (carrega texto do arquivo) e um botão 'Cor' (muda a cor da fonte).
