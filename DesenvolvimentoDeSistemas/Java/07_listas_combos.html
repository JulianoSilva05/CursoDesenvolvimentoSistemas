Listas Dinâmicas e Modelos de Dados

Conceitos Fundamentais (Teoria)

DefaultListModel

Permite adicionar, remover e alterar itens da lista em tempo de execução.

SelectionMode

Define se o usuário pode escolher um item (SINGLE) ou vários (MULTIPLE).

ListCellRenderer

Permite customizar visualmente cada item da lista (cores, ícones).

JSpinner

Alternativa ao ComboBox para números ou datas sequenciais.

Aprofundamento Técnico

1. Manipulação Dinâmica

Diferente de arrays estáticos, o DefaultListModel funciona como um ArrayList. Você pode usar model.addElement() ou model.remove() a qualquer momento.

2. Renderizadores (Avançado)

Você pode fazer uma JList exibir painéis inteiros em vez de apenas strings, implementando a interface ListCellRenderer.

Exemplos de Implementação

Analise o código abaixo para entender a aplicação:

// Modelo Dinâmico DefaultListModel modelo = new DefaultListModel<>(); modelo.addElement("Tarefa 1"); modelo.addElement("Tarefa 2"); JList lista = new JList<>(modelo); lista.setSelectionMode(ListSelectionModel.SINGLE_SELECTION); // Botão para remover selecionado btnDeletar.addActionListener(e -> { int index = lista.getSelectedIndex(); if(index != -1) modelo.remove(index); });

Atividade Prática (1 Hora)

Desafio: Crie um gerenciador de 'Lista de Compras'. Tenha um campo de texto e um botão 'Adicionar' que insere itens na JList. Adicione também um botão 'Remover' que apaga o item selecionado.
