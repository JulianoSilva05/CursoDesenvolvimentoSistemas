Atualizando do AWT para Swing

A Regra do "J"

Muitos componentes do Swing são evoluções diretas do AWT. Para usá-los, geralmente basta adicionar um "J" na frente do nome da classe.

  • Button → JButton
  • TextField → JTextField
  • Label → JLabel
  • Frame → JFrame

Importando

Enquanto o AWT fica em java.awt.*, o Swing fica em:

import javax.swing.*;

Seu Primeiro Componente Swing

Vamos criar um botão simples. Note que o JButton pode ter ícones, algo que o botão do AWT não fazia facilmente.

import javax.swing.*; import java.awt.*; public class HelloSwing { public static void main(String[] args) { // 1. Criar a Janela (JFrame) JFrame frame = new JFrame("Minha Primeira Janela"); frame.setSize(300, 200); frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE); // Importante! // 2. Criar o Componente (JButton) JButton botao = new JButton("Clique em Mim!"); // 3. Adicionar à Janela frame.add(botao); // 4. Mostrar frame.setVisible(true); } }
Dica: O setDefaultCloseOperation é crucial. Sem ele, fechar a janela apenas a esconde, mas o programa continua rodando em segundo plano.

Estrutura de uma Janela (JFrame)

O JFrame não é apenas um retângulo vazio. Ele tem camadas.

Content Pane

É onde você coloca seus botões, textos e imagens. Você raramente adiciona coisas direto no JFrame, mas sim no seu Content Pane.

frame.getContentPane().add(botao);

(Nas versões novas do Java, frame.add() faz isso automaticamente para você).

JPanel

Para organizar vários componentes, usamos JPanel. É como uma "div" no HTML.

Você agrupa botões num Painel, e coloca o Painel na Janela.

Um Exemplo Mais Rico

Vamos criar uma janela com texto e imagem.

// Ícone (certifique-se de ter a imagem) ImageIcon icone = new ImageIcon("logo_senai.png"); // Label com Texto e Ícone JLabel etiqueta = new JLabel("Bem-vindo ao Curso!", icone, JLabel.CENTER); // Configurando posição do texto em relação à imagem etiqueta.setHorizontalTextPosition(JLabel.CENTER); etiqueta.setVerticalTextPosition(JLabel.BOTTOM); frame.add(etiqueta);

Isso demonstra o poder do Swing: componentes híbridos (texto + imagem) fáceis de usar.

Atividade Prática 01

  2. Tamanho 400x300.
