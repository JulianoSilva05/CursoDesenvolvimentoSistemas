Conceitos Fundamentais (Teoria)

JFrame

O container principal de nível superior (Janela).

Tamanho

setSize(w, h) define tamanho fixo. pack() ajusta ao conteúdo.

Visibilidade

Janelas nascem invisíveis. Use setVisible(true).

Fechamento

setDefaultCloseOperation define o que acontece ao clicar no X.

Aprofundamento Técnico

1. ContentPane

Você não adiciona componentes diretamente ao JFrame, mas sim ao seu ContentPane.

frame.getContentPane().add(botao);

Nas versões novas do Java, frame.add() é um atalho para isso.

2. Centralizando a Janela

Use setLocationRelativeTo(null) após definir o tamanho para centralizar na tela.

Exemplos de Implementação

Analise o código abaixo para entender a aplicação:

JFrame frame = new JFrame("Minha Janela"); frame.setSize(400, 300); frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE); frame.setLocationRelativeTo(null); // Centraliza JButton btn = new JButton("Oi"); frame.add(btn); // Adiciona ao centro (BorderLayout padrão) frame.setVisible(true);
