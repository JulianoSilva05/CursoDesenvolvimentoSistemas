Aula 14: Bordas e Estilização

BorderFactory, CompoundBorder

Aula 14

Duração: 3 Horas (2h Teoria + 1h Prática)

Refinando o Visual da Interface

Conceitos Fundamentais (Teoria)

CompoundBorder

Combina duas bordas (ex: uma Borda Externa decorativa + uma Borda Interna vazia para margem).

MatteBorder

Borda sólida ou com ícones repetidos.

LineBorder

Borda simples de cor e espessura definidas.

SoftBevelBorder

Versão mais suave do BevelBorder (relevo).

Aprofundamento Técnico

1. Margens com Bordas

No Swing, não existe 'margin' como no CSS. Para dar espaço interno, usamos EmptyBorder.

2. Bordas Compostas

Para ter uma borda visual E um espaço, usamos CompoundBorder.

Border linha = BorderFactory.createLineBorder(Color.BLACK); Border espaco = BorderFactory.createEmptyBorder(10, 10, 10, 10); painel.setBorder(BorderFactory.createCompoundBorder(linha, espaco));

Exemplos de Implementação

Analise o código abaixo para entender a aplicação:

JPanel p = new JPanel(); // Borda com Título e alinhamento TitledBorder titulo = BorderFactory.createTitledBorder("Dados Pessoais"); titulo.setTitleJustification(TitledBorder.CENTER); p.setBorder(titulo);

Atividade Prática (1 Hora)

Desafio: Crie um painel de 'Login' bonito. Use CompoundBorder para dar margem interna aos campos e uma TitledBorder externa. Use cores nas bordas.
