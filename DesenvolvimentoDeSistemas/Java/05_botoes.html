Aula 05: Botões e Eventos Complexos

JButton, Toggle, Radio, CheckBox

Aula 05

Duração: 3 Horas (2h Teoria + 1h Prática)

Interação Avançada com o Usuário

Conceitos Fundamentais (Teoria)

ButtonGroup

Agrupa RadioButtons para permitir apenas uma seleção por vez.

ItemListener

Evento específico para detectar mudança de estado (selecionado/não).

Mnemônicos

Teclas de atalho (Alt+X) para acionar botões via teclado.

Ícones de Estado

Botões podem ter ícones diferentes para 'passar o mouse' ou 'pressionado'.

Aprofundamento Técnico

1. ButtonGroup (Exclusividade)

Para criar opções mutuamente exclusivas (como Gênero ou Pagamento), você deve adicionar os RadioButtons a um ButtonGroup lógico.

2. Customização de Ícones

Um botão pode mudar de aparência dinamicamente.

btn.setRolloverIcon(new ImageIcon("hover.png")); btn.setPressedIcon(new ImageIcon("click.png"));

Exemplos de Implementação

Analise o código abaixo para entender a aplicação:

// Grupo de Opções JRadioButton r1 = new JRadioButton("Crédito"); JRadioButton r2 = new JRadioButton("Débito"); ButtonGroup grupo = new ButtonGroup(); grupo.add(r1); grupo.add(r2); // Agora só um pode ser marcado // CheckBox com Evento de Item JCheckBox chk = new JCheckBox("Habilitar Notificações"); chk.addItemListener(e -> { if(e.getStateChange() == ItemEvent.SELECTED) { System.out.println("Ligado"); } });

Atividade Prática (1 Hora)

Desafio: Crie um formulário de pedido de Pizza. Use CheckBoxes para ingredientes (múltiplos) e RadioButtons para o tamanho (P/M/G - exclusivo). Adicione um botão 'Pedir' que lista o que foi escolhido.
