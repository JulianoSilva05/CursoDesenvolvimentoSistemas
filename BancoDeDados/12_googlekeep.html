Curso de Desenvolvimento de Sistemas - SENAI

Duração estimada: 4 horas (2h Teoria + 2h Prática)

O que é o Google Keep?

É uma ferramenta gratuita de criação de notas do Google.

Funcionalidades:

  • Criar notas rápidas (texto, áudio, foto).
  • Criar listas de tarefas (checklists).
  • Definir lembretes (baseados em hora ou localização!).
  • Sincronização na nuvem (PC, Android, iOS).

Por que usar?

A memória humana é falha. O Keep serve como um "segundo cérebro" para captura rápida.

  • Teve uma ideia no ônibus? Anote.
  • Lista de compras? Checklist.
  • Precisa lembrar de algo ao chegar na escola? Lembrete por local.

Organizando suas Notas

Para não virar bagunça:

  • Cores: Use cores para categorias (ex: Vermelho para Urgente, Verde para Pessoal).
  • Marcadores (Labels): Tags como #Estudos, #Trabalho, #Projetos.
  • Fixar: Mantenha as notas mais importantes no topo.

Colaboração

Você pode compartilhar notas com outras pessoas.

Útil para:

  • Listas de tarefas em grupo.
  • Brainstorming de projetos.
  • Lista de compras compartilhada com a família.

Integração com Google Docs

Você pode arrastar suas notas do Keep diretamente para um documento do Google Docs.

Isso é ótimo para transformar rascunhos rápidos em trabalhos formais.

Atividade Prática (2 Horas)

Organização Total

Parte 1: Plano de Estudos (40 min)

Crie uma nota tipo "Checklist" no Google Keep chamada "Plano de Estudos - Banco de Dados".

Liste 10 tópicos que aprendemos (ex: JOIN, DDL, Normalização) e marque os que você já domina.

Parte 2: Lembretes Inteligentes (40 min)

Crie uma nota com lembrete de localização: "Revisar SQL ao chegar em casa". (Simule ou descreva como fez).

Parte 3: Kanban Pessoal (40 min)

Use o sistema de cores para criar um Kanban simples no Keep:

  • Nota Vermelha: "A Fazer" (Liste 3 tarefas pendentes).
  • Nota Amarela: "Fazendo" (O que está estudando agora).
  • Nota Verde: "Feito" (O que já concluiu).
