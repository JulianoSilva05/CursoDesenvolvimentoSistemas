O que é SQL?

SQL (Structured Query Language) é a linguagem padrão para interagir com bancos de dados relacionais.

Foco de hoje: DML (Data Manipulation Language).

  • SELECT: Ler dados.
  • INSERT: Criar dados.
  • UPDATE: Atualizar dados.
  • DELETE: Apagar dados.

INSERT INTO

Adiciona novos registros.

INSERT INTO Clientes (Nome, Email, Idade) VALUES ('João Silva', 'joao@email.com', 30);
Sempre liste as colunas para evitar erros futuros!

SELECT (O Coração do SQL)

Recupera dados de uma ou mais tabelas.

-- Selecionar tudo SELECT * FROM Clientes; -- Selecionar colunas específicas SELECT Nome, Email FROM Clientes;

WHERE (O Filtro)

Essencial para não trazer (ou alterar) dados indesejados.

SELECT * FROM Clientes WHERE Idade > 18; SELECT * FROM Clientes WHERE Cidade = 'São Paulo' AND Ativo = TRUE;

ORDER BY

Organiza a bagunça.

SELECT * FROM Produtos ORDER BY Preco ASC; -- Menor pro maior SELECT * FROM Produtos ORDER BY DataCadastro DESC; -- Mais recente primeiro

UPDATE

Atualiza registros.

UPDATE Clientes SET Email = 'novo@email.com', Telefone = '9999-8888' WHERE ID = 10;
SEM WHERE = DEMISSÃO! (Atualiza a tabela toda).

DELETE

Remove registros.

DELETE FROM Clientes WHERE ID = 5;
SEM WHERE = DEMISSÃO! (Apaga a tabela toda).

LIKE (Busca de Texto)

  • %: Qualquer coisa.
  • _: Um caractere.
SELECT * FROM Clientes WHERE Nome LIKE 'A%'; -- Começa com A SELECT * FROM Clientes WHERE Email LIKE '%@gmail.com'; -- Termina com gmail

Atividade Prática (2 Horas)

Gerenciando uma Locadora

Considere a tabela Filmes (ID, Titulo, Genero, Ano, PrecoAluguel, Disponivel).

Parte 1: Inserção (40 min)

Escreva os comandos para inserir 5 filmes variados (Ação, Comédia, Terror).

Parte 2: Consultas (40 min)

  1. Listar todos os filmes de 'Comédia'.
  2. Listar filmes lançados após 2020.
  3. Listar filmes com preço menor que 10.00, ordenados pelo título.

Parte 3: Alteração e Remoção (40 min)

  1. Aumentar o preço de todos os filmes de 'Ação' em 10%.
  2. Marcar como indisponível (Disponivel = FALSE) o filme de ID 3.
  3. Deleter todos os filmes anteriores a 1990.
