O que é SQL?

SQL (Structured Query Language) é a linguagem padrão para interagir com bancos de dados relacionais.

Hoje vamos focar no DML (Data Manipulation Language), ou seja, como manipular os dados dentro das tabelas.

  • SELECT: Ler dados.
  • INSERT: Criar dados.
  • UPDATE: Atualizar dados.
  • DELETE: Apagar dados.

INSERT INTO (Inserindo Dados)

O comando INSERT adiciona novas linhas (registros) a uma tabela.

-- Sintaxe Completa (Recomendada) INSERT INTO Clientes (Nome, Email, Idade) VALUES ('João Silva', 'joao@email.com', 30); -- Sintaxe Resumida (Cuidado: depende da ordem das colunas) INSERT INTO Clientes VALUES ('Maria Souza', 'maria@email.com', 25);
Sempre prefira listar as colunas. Isso evita erros se a estrutura da tabela mudar no futuro.

SELECT (Consultando Dados)

É o comando mais utilizado. Permite recuperar dados de uma ou mais tabelas.

-- Selecionar TUDO (todas as colunas e linhas) SELECT * FROM Clientes; -- Selecionar colunas específicas (Melhor performance) SELECT Nome, Email FROM Clientes;

WHERE (O Filtro)

O WHERE define condições para filtrar os resultados. Funciona no SELECT, UPDATE e DELETE.

-- Clientes com mais de 18 anos SELECT * FROM Clientes WHERE Idade > 18; -- Clientes com nome específico SELECT * FROM Clientes WHERE Nome = 'João Silva'; -- Uso de Operadores Lógicos (AND, OR) SELECT * FROM Clientes WHERE Idade > 18 AND Cidade = 'São Paulo';

ORDER BY (Ordenando Resultados)

Organiza a saída dos dados.

  • ASC: Ascendente (A-Z, 0-9). Padrão se omitido.
  • DESC: Descendente (Z-A, 9-0).
-- Ordenar por Nome (A-Z) SELECT * FROM Clientes ORDER BY Nome; -- Ordenar por Idade (Mais velho para mais novo) SELECT * FROM Clientes ORDER BY Idade DESC;

UPDATE (Alterando Dados)

Modifica registros existentes.

-- Mudar o email do João UPDATE Clientes SET Email = 'joao.novo@email.com' WHERE ID = 1; -- Aumentar o preço de todos os produtos em 10% UPDATE Produtos SET Preco = Preco * 1.10;
PERIGO: Se você esquecer o WHERE, vai alterar TODOS os registros da tabela!

DELETE (Removendo Dados)

Remove registros da tabela.

-- Apagar o cliente com ID 5 DELETE FROM Clientes WHERE ID = 5; -- Apagar todos os clientes inativos DELETE FROM Clientes WHERE Ativo = FALSE;
PERIGO EXTREMO: DELETE FROM Clientes; (sem WHERE) apaga a tabela inteira!

Funções de Agregação

SQL pode calcular coisas para nós:

  • COUNT(*): Conta registros.
  • SUM(coluna): Soma valores.
  • AVG(coluna): Média aritmética.
  • MAX(coluna) / MIN(coluna): Maior e menor valor.
-- Quantos clientes temos? SELECT COUNT(*) FROM Clientes; -- Qual a média de preço dos produtos? SELECT AVG(Preco) FROM Produtos;

LIKE (Busca Inteligente)

Usado para buscar partes de um texto.

  • %: Representa qualquer sequência de caracteres.
  • _: Representa um único caractere.
-- Nomes que COMEÇAM com 'Ana' (Ana, Ana Paula, Anabel) SELECT * FROM Clientes WHERE Nome LIKE 'Ana%'; -- Nomes que TERMINAM com 'Silva' SELECT * FROM Clientes WHERE Nome LIKE '%Silva'; -- Nomes que CONTÉM 'Oliveira' em qualquer lugar SELECT * FROM Clientes WHERE Nome LIKE '%Oliveira%';
